# Spam Email Detection using Truncated SVD and WKNN

## 📌 Objective
Classify emails as spam or legitimate using Machine Learning.

## ⚙️ Methodology
- Text Preprocessing
- TF-IDF Vectorization
- Truncated SVD (Dimensionality Reduction)
- Weighted KNN (WKNN) Classification

## 📂 Dataset
Spam Email Dataset (CSV format)

## 🚀 How to Run

1. Clone repository:
   git clone https://github.com/your-username/spam-email-detection.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the project:
   cd src
   python main.py
